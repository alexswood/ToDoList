## General info
The purpose of this project is to create a small application to allow a user to
write and manage ToDo items.
For the first release the user will be able to create a ToDo item with a title
and description, they will also be able to edit, delete and view ToDo items.
The second release will provide functionality to add a priority.
The third release will provide functionality to add a due date.

## Technologies
* Visual Studio 2019

## Setup
No pre-requisites required.

## Features
Release 1
* Add a ToDo item
* View a ToDo item
* Remove a ToDo item
* Edit a ToDo item

Release 2
* Add a priority for a ToDo item

Release 3
* Add a due date for a ToDo item
